## Description 
back-end server with nestJs for 
<a href="https://github.com/IHesamI/car-sales-client" target="_blank">online car sales platforms</a>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash

# unit tests with html docs
# Notice! In users.service.spec.ts file put password for testing purpose where it has comment.

$ npm run test:api

# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
